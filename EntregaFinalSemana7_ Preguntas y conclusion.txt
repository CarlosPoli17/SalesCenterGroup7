Las dificultades presentadas durante el desarrollo del proyecto.  

 

La experiencia en desarrollo de los tres integrantes de la entrega facilito la ejecución del proyecto, ya que el nivel de conocimiento del lenguaje JAVA es alto. 

Entre las dificultades más relevantes estuvieron: 

Entender cómo funcionaban las interfaces (BufferedReader / BufferedWriter) funcionalmente para insertar cabeceras, datos y saltos de línea. 

Manejar la agregación de datos para manejar ArrayList dentro de otro ArrayList e instancias de clase dentro de ArrayList; entender como acceder a los datos dentro de los ArrayList, iterar con ellos y modificarlos.  

Dada la complejidad que fue tomando el código, ordenar el informe de ventas final tuvo un nivel de dificultad alta, ya que no podíamos emplear los métodos convencionales como ArrayList.sort() porque el criterio de ordenación era un atributo dentro de los elementos del ArrayList, así que, tuvimos que ingeniar la forma de buscar el mayor valor de ventas del ArrayList, guardarlo en un nuevo ArrayList y eliminarlo, esto iterativamente hasta que quedara el último elemento (el menor valor). 

Implementar desde el principio el manejo de errores fue fundamental para que al final se facilitara implementar las funcionalidades de validación opcionales, ya que de otro modo hubiera sido complejo modificar el código.
